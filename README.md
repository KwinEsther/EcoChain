# EcoChain

A decentralized environmental action rewards platform built on Stacks blockchain. Incentivizes environmental activism through token rewards for consistent eco-friendly actions and impact streaks.

## Features
- Environmental action tracking and validation
- Eco token rewards with impact-based bonuses
- Sustainability pledge mechanism for long-term commitment
- Transparent environmental impact metrics

## Usage
Deploy with Clarinet for decentralized environmental action tracking and token distribution.
